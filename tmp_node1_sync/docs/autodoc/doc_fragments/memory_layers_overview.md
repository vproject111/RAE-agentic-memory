# 4-Layer Memory Architecture

RAE implements a **complete 4-layer cognitive architecture** inspired by human memory systems:

```
Layer 4: Reflective Memory (Meta-Learnings)
  ↓
Layer 3: Long-Term Memory (Facts + Events + Profiles)
  ↓
Layer 2: Working Memory (Active Task Context)
  ↓
Layer 1: Sensory Memory (Raw Inputs)
```

Each layer serves a distinct cognitive function, from immediate sensory input to high-level meta-learning.

**See**: [Complete Memory Layers Documentation](../../architecture/MEMORY_LAYERS.md)
